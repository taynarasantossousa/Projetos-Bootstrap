# Página de Produto SaaS

Projeto desenvolvido com **HTML5 + Bootstrap 5**, com o objetivo de criar uma Landing Page moderna para um software fictício (Software as a Service).

Este projeto foca no aprendizado do **Grid System do Bootstrap**, tipografia, botões e utilitários de espaçamento.

---

## 📌 Nível

Básico – Introdução ao Bootstrap

---

## 🎯 Objetivo do projeto

Criar uma página de produto para um software fictício, usando apenas classes do Bootstrap, sem CSS manual.

A página simula uma Landing Page para venda de um aplicativo de tarefas.

---

## 🧩 Componentes utilizados

### ✅ Grid System

Utilizado para criar layout responsivo.

Classes usadas:

* container
* row
* col-12
* col-md-6
* col-md-4

O layout se adapta automaticamente para celular.

---

### ✅ Buttons

Botões do Bootstrap:

* btn
* btn-primary
* btn-outline-dark
* btn-lg

Usados para ações como:

* Começar
* Criar conta
* Saiba mais

---

### ✅ Typography

Classes usadas para texto:

* display-4
* lead
* text-muted
* text-center

Usado para títulos e descrição.

---

### ✅ Spacing Utilities

Espaçamentos sem CSS manual:

* mt-5
* mb-4
* py-5
* gap-3

Usado para organizar o layout.

---

## 🛠 Tecnologias usadas

* HTML5
* Bootstrap 5 (CDN)

Bootstrap usado via CDN:

https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css

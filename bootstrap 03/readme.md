# TravelGo • Agência de Viagens

Projeto desenvolvido com **HTML5 + Bootstrap 5 + CSS**, com o objetivo de criar um site moderno para uma agência de viagens.

O site possui banner com carousel, cards de pacotes, seção promocional e layout responsivo.

---

## 📌 Nível

Intermediário

---

## 🎯 Objetivo do projeto

Criar um site completo usando Bootstrap, aplicando:

* Navbar responsiva
* Carousel (banner)
* Cards com pacotes
* Badges de preço
* Botões
* Seção promocional
* Footer
* Responsividade

---

## 🧩 Componentes utilizados

### ✅ Navbar

Menu responsivo com botão para mobile.

Classes usadas:

* navbar
* navbar-expand-lg
* navbar-dark
* bg-dark
* navbar-toggler
* collapse

---

### ✅ Carousel

Banner com imagens automáticas.

Classes usadas:

* carousel
* carousel-inner
* carousel-item
* carousel-control-prev
* carousel-control-next

Usado para mostrar banners da agência.

---

### ✅ Grid System

Usado para organizar os cards.

Classes usadas:

* container
* row
* col-12
* col-md-4
* g-4

Layout responsivo:

* 1 card no celular
* 3 cards no desktop

---

### ✅ Cards

Usado para mostrar pacotes.

Classes usadas:

* card
* card-body
* card-footer
* card-img-top
* shadow

---

### ✅ Badges

Usado para mostrar preços.

Classes usadas:

* badge
* bg-danger
* bg-success
* bg-warning

---

### ✅ Buttons

Botões de ação.

Classes usadas:

* btn
* btn-primary
* btn-warning
* btn-lg
* w-100

---

### ✅ Utilities

Classes usadas:

* text-center
* py-5
* mb-5
* fw-bold
* d-none
* d-md-block

Usadas para responsividade e espaçamento.

---

### ✅ Promo Section

Seção promocional visível apenas no desktop.

Classe usada:

* d-none d-md-block

---

### ✅ Footer

Rodapé com fundo escuro.

Classes usadas:

* bg-dark
* text-white
* text-center
* py-4

# Projetos Bootstrap

Este repositório contém 5 projetos desenvolvidos com **HTML5 + Bootstrap 5**, com o objetivo de praticar layout responsivo, componentes e utilitários do Bootstrap.

Os projetos vão do nível básico ao avançado.

---

## 📌 Tecnologias usadas

* HTML5
* Bootstrap 5
* CSS
* Bootstrap Icons

CDN Bootstrap:

https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css

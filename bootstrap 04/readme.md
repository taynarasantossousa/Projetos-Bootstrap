# Blog Tech & Help Center

Projeto desenvolvido com **HTML5 + Bootstrap 5**, com o objetivo de criar um portal de notícias de tecnologia que também funciona como central de ajuda (FAQ).

O projeto utiliza apenas recursos do Bootstrap, sem JavaScript manual.

---

## 📌 Nível

Intermediário / Avançado

---

## 🎯 Objetivo do projeto

Criar um site com:

* Blog de tecnologia
* Central de ajuda (FAQ)
* Modais de login e newsletter
* Navegação entre páginas
* Feedback visual com alerts e badges

Tudo usando classes e atributos do Bootstrap.

---

## 🧩 Componentes utilizados

### ✅ Navbar

Menu responsivo com botão mobile.

Classes usadas:

* navbar
* navbar-expand-lg
* navbar-dark
* bg-dark
* collapse

---

### ✅ Alerts

Mensagem informando novo post.

Classes usadas:

* alert
* alert-info

---

### ✅ Cards

Usado para mostrar notícias.

Classes usadas:

* card
* card-body
* card-title
* badge

---

### ✅ Badges

Usado para categoria:

* bg-primary
* bg-success
* bg-warning

---

### ✅ Pagination

Usado para navegar entre páginas.

Classes usadas:

* pagination
* page-item
* page-link

---

### ✅ Accordion (FAQ)

Perguntas frequentes.

Classes usadas:

* accordion
* accordion-item
* accordion-button
* collapse

Controlado com:

data-bs-toggle
data-bs-target

---

### ✅ Modal

Usado para:

* Login
* Newsletter

Classes usadas:

* modal
* modal-dialog
* modal-content
* modal-header
* modal-body
* modal-footer

Controlado com:

data-bs-toggle
data-bs-target

---

### ✅ Data Attributes

Usado para controlar Bootstrap sem JS.

Exemplos:

* data-bs-toggle="modal"
* data-bs-toggle="collapse"
* data-bs-target="#login"

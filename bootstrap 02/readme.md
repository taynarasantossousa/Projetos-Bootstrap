# Admin Dashboard

Projeto desenvolvido com **HTML5, Bootstrap 5, Bootstrap Icons e CSS**, com o objetivo de criar um painel administrativo moderno e responsivo.

Este projeto simula um sistema de administração com menu lateral, cards de estatísticas e tabela de usuários.

---

## 📌 Nível

Intermediário / Avançado

---

## 🎯 Objetivo do projeto

Criar um painel administrativo (Dashboard) usando Bootstrap, com layout profissional parecido com sistemas reais.

O projeto possui:

* Sidebar lateral
* Navbar superior
* Cards com informações
* Tabela responsiva
* Badges de status
* Ícones
* CSS personalizado

---

## 🧩 Componentes utilizados

### ✅ Grid System

Utilizado para organizar os cards e o layout.

Classes usadas:

* container-fluid
* row
* col-12
* col-md-4
* g-4

---

### ✅ Navbar

Barra superior com título do painel.

Classes usadas:

* navbar
* bg-white
* shadow-sm
* px-4

---

### ✅ Sidebar

Menu lateral com navegação.

Componentes usados:

* nav

* nav-item

Ícones usando Bootstrap Icons.

### ✅ Cards

Cards para mostrar estatísticas.

# Checkout de E-commerce

Projeto desenvolvido com **HTML5 + Bootstrap 5 + CSS**, com o objetivo de criar uma página de finalização de compra (checkout) de uma loja virtual.

O projeto utiliza formulários avançados, validação visual, resumo do carrinho e customização do Bootstrap.

---

## 📌 Nível

Avançado

---

## 🎯 Objetivo do projeto

Criar uma página de checkout com layout profissional usando Bootstrap, aplicando:

* Formulários avançados
* Floating labels
* Input group
* Validação visual
* List group
* Badges
* Grid system
* Customização com CSS

---

## 🧩 Componentes utilizados

### ✅ Grid System

Usado para dividir formulário e carrinho.

Classes usadas:

* container
* row
* col-md-7
* col-md-5

Layout:

* Formulário à esquerda
* Carrinho à direita
